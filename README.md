# Assignment 3: Sorting and Searching Algorithm Analysis System
This project is about sorting and searching algorithms.  
Basically, I had to choose three algorithms, implement them in Java, test them on different arrays, and compare their execution time.

I chuse:

- Selection Sort — basic sorting algorithm
Merge Sort — advanced sorting algorithm
- Binary Search — searching algorithm

The main goal was to see which algorithm is faster and how array size affects performance.

In simple words:  
Selection Sort is like a weak anime side character who tries hard but takes too long.  
Merge Sort is like the main character after training arc.  
Binary Search is that overpowered character who finds the enemy in 2 seconds, but only if the map is organized.
